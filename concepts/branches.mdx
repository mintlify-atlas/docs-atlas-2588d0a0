OpenCBS Cloud supports multi-branch operations, enabling microfinance institutions to manage multiple locations from a single system. Each branch operates independently while sharing centralized configuration and reporting.

## Branch Concept

A **branch** represents a physical or organizational location where the MFI conducts business. Branches serve as the primary organizational unit for:

- Client profiles
- Loan portfolios
- User assignments
- Accounting entries
- Reporting segments
- Permission boundaries

<Info>
Every profile, user, and account in OpenCBS must be associated with a branch. This ensures proper segregation and enables branch-level reporting.
</Info>

## Branch Service

### Service Implementation

```java
package com.opencbs.core.services;

@Service
public class BranchService {
    
    private final BranchRepository branchRepository;
    
    @Autowired
    public BranchService(BranchRepository branchRepository) {
        this.branchRepository = branchRepository;
    }
    
    @Transactional
    public Branch create(Branch branch) {
        this.updateBranchCustomFields(branch);
        branch.setId(null);
        return this.branchRepository.save(branch);
    }
    
    @Transactional
    public Branch update(Branch branch) {
        this.updateBranchCustomFields(branch);
        return this.branchRepository.save(branch);
    }
    
    public Optional<Branch> findByName(String name) {
        return this.branchRepository.findByName(name);
    }
    
    public Optional<Branch> findOne(Long id) {
        return Optional.of(this.branchRepository.findOne(id));
    }
    
    public Page<Branch> findAll(Pageable pageable) {
        return this.branchRepository.findAll(pageable);
    }
    
    public Page<Branch> findByNameContaining(Pageable pageable, String search) {
        if (search == null) search = "";
        return this.branchRepository.getBySearchPattern(pageable, search);
    }
    
    public List<Branch> findAll() {
        return this.branchRepository.findAll();
    }
    
    private void updateBranchCustomFields(Branch branch) {
        this.getBranchCustomFields(branch)
                .forEach(x -> {
                    x.setCreatedBy(UserHelper.getCurrentUser());
                    x.setCreatedAt(DateHelper.getLocalDateTimeNow());
                    x.setVerifiedBy(UserHelper.getCurrentUser());
                    x.setVerifiedAt(DateHelper.getLocalDateTimeNow());
                    x.setStatus(EntityStatus.LIVE);
                });
    }
    
    private List<BranchCustomFieldValue> getBranchCustomFields(Branch branch) {
        return branch.getCustomFieldValues();
    }
}
```

### Branch-Level Permissions

Control user access by branch:

```java
@Service
public class ProfileSecurityService {
    
    public boolean canAccessProfile(User user, Profile profile) {
        // Standard users can only access profiles in their branch
        if (user.getRole().getName().equals("BRANCH_MANAGER")) {
            return user.getBranch().getId().equals(profile.getBranch().getId());
        }
        
        // Admin users can access all branches
        if (user.getRole().getName().equals("ADMIN")) {
            return true;
        }
        
        return false;
    }
}
```

### Cross-Branch Transactions

Inter-branch transfers require special handling:

```java
@Service
public class TransferService {
    
    @Transactional
    public void transferBetweenBranches(
            Account sourceBranch,
            Account targetBranch,
            BigDecimal amount,
            String description) {
        
        // Debit source branch
        AccountingEntry debitEntry = new AccountingEntry();
        debitEntry.setAccount(sourceBranch);
        debitEntry.setDebitAmount(amount);
        debitEntry.setDescription(description);
        debitEntry.setBranch(sourceBranch.getBranch());
        
        // Credit target branch
        AccountingEntry creditEntry = new AccountingEntry();
        creditEntry.setAccount(targetBranch);
        creditEntry.setCreditAmount(amount);
        creditEntry.setDescription(description);
        creditEntry.setBranch(targetBranch.getBranch());
        
        accountingEntryService.createEntries(
            Arrays.asList(debitEntry, creditEntry)
        );
    }
}
```

### Branch Consolidation

Aggregate data across all branches:

```java
@Service
public class ConsolidatedReportService {
    
    public BranchPerformanceReport generateConsolidatedReport(
            LocalDate startDate, 
            LocalDate endDate) {
        
        List<Branch> allBranches = branchService.findAll();
        BranchPerformanceReport consolidated = new BranchPerformanceReport();
        
        for (Branch branch : allBranches) {
            BranchMetrics metrics = calculateBranchMetrics(
                branch, startDate, endDate
            );
            consolidated.addBranchMetrics(branch.getName(), metrics);
        }
        
        return consolidated;
    }
    
    private BranchMetrics calculateBranchMetrics(
            Branch branch,
            LocalDate startDate,
            LocalDate endDate) {
        
        BranchMetrics metrics = new BranchMetrics();
        
        // Active clients
        metrics.setActiveClients(
            profileRepository.countByBranchAndStatus(
                branch, EntityStatus.LIVE
            )
        );
        
        // Loan portfolio
        metrics.setLoanPortfolio(
            loanRepository.sumOutstandingBalanceByBranch(branch)
        );
        
        // Savings deposits
        metrics.setSavingsDeposits(
            savingRepository.sumBalanceByBranch(branch)
        );
        
        // Disbursements
        metrics.setDisbursements(
            loanEventRepository.sumDisbursementsByBranchAndDateRange(
                branch, startDate, endDate
            )
        );
        
        return metrics;
    }
}
```

## Branch Configuration

### Creating a Branch

```java
// Backend service
@Service
public class BranchConfigurationService {
    
    @Transactional
    public Branch setupNewBranch(BranchSetupDto dto) {
        // 1. Create branch entity
        Branch branch = new Branch();
        branch.setName(dto.getName());
        
        // 2. Add custom field values
        for (CustomFieldValueDto fieldValue : dto.getCustomFields()) {
            BranchCustomFieldValue value = new BranchCustomFieldValue();
            value.setCustomField(
                customFieldService.findByName(fieldValue.getFieldName())
            );
            value.setValue(fieldValue.getValue());
            value.setStatus(EntityStatus.LIVE);
            value.setOwner(branch);
            branch.getCustomFieldValues().add(value);
        }
        
        // 3. Save branch
        Branch savedBranch = branchService.create(branch);
        
        // 4. Create default accounts for branch
        createDefaultBranchAccounts(savedBranch);
        
        // 5. Assign initial users
        assignBranchManager(savedBranch, dto.getManagerUserId());
        
        return savedBranch;
    }
    
    private void createDefaultBranchAccounts(Branch branch) {
        // Cash account
        Account cashAccount = new Account();
        cashAccount.setName(branch.getName() + " - Cash");
        cashAccount.setNumber("1100-" + branch.getId());
        cashAccount.setIsDebit(true);
        cashAccount.setBranch(branch);
        accountService.create(cashAccount);
        
        // Loan portfolio account
        Account loanAccount = new Account();
        loanAccount.setName(branch.getName() + " - Loan Portfolio");
        loanAccount.setNumber("1300-" + branch.getId());
        loanAccount.setIsDebit(true);
        loanAccount.setBranch(branch);
        accountService.create(loanAccount);
    }
}
```

### Branch REST Endpoint

```java
@RestController
@RequestMapping("/api/branches")
public class BranchController extends BaseController {
    
    private final BranchService branchService;
    private final BranchMapper branchMapper;
    
    @GetMapping
    public Page<BranchDto> getBranches(
            @RequestParam(required = false) String search,
            Pageable pageable) {
        
        Page<Branch> branches = search != null 
            ? branchService.findByNameContaining(pageable, search)
            : branchService.findAll(pageable);
        
        return branches.map(branchMapper::toDto);
    }
    
    @GetMapping("/{id}")
    public ResponseEntity<BranchDto> getBranch(@PathVariable Long id) {
        return branchService.findOne(id)
            .map(branchMapper::toDto)
            .map(ResponseEntity::ok)
            .orElse(ResponseEntity.notFound().build());
    }
    
    @PostMapping
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<BranchDto> createBranch(@RequestBody BranchDto dto) {
        Branch branch = branchMapper.toEntity(dto);
        Branch created = branchService.create(branch);
        return ResponseEntity.created(
            URI.create("/api/branches/" + created.getId())
        ).body(branchMapper.toDto(created));
    }
    
    @PutMapping("/{id}")
    @PreAuthorize("hasRole('ADMIN')")
    public ResponseEntity<BranchDto> updateBranch(
            @PathVariable Long id,
            @RequestBody BranchDto dto) {
        
        return branchService.findOne(id)
            .map(existing -> {
                branchMapper.updateEntity(dto, existing);
                Branch updated = branchService.update(existing);
                return ResponseEntity.ok(branchMapper.toDto(updated));
            })
            .orElse(ResponseEntity.notFound().build());
    }
}
```

```java
@Service
public class BranchComparisonService {
    
    public List<BranchComparison> compareBranches(
            LocalDate reportDate) {
        
        return branchService.findAll().stream()
            .map(branch -> {
                BranchComparison comparison = new BranchComparison();
                comparison.setBranchName(branch.getName());
                
                // Client metrics
                comparison.setActiveClients(
                    profileRepository.countActiveByBranch(branch, reportDate)
                );
                
                // Portfolio metrics
                comparison.setLoanPortfolio(
                    loanRepository.sumOutstandingByBranch(branch, reportDate)
                );
                comparison.setParRate(
                    calculatePAR(branch, reportDate)
                );
                
                // Financial metrics
                comparison.setNetIncome(
                    calculateNetIncome(branch, reportDate)
                );
                
                // Efficiency metrics
                comparison.setOperatingExpenseRatio(
                    calculateOER(branch, reportDate)
                );
                
                return comparison;
            })
            .sorted(Comparator.comparing(BranchComparison::getLoanPortfolio).reversed())
            .collect(Collectors.toList());
    }
}
```

### Branch Profitability

```java
public class BranchProfitabilityCalculator {
    
    public BranchPL calculateProfitAndLoss(
            Branch branch,
            LocalDate startDate,
            LocalDate endDate) {
        
        BranchPL pl = new BranchPL();
        
        // Revenue
        BigDecimal interestIncome = accountingEntryRepository
            .sumCreditsByAccountTypeAndBranch(
                AccountType.INTEREST_INCOME, branch, startDate, endDate
            );
        BigDecimal feeIncome = accountingEntryRepository
            .sumCreditsByAccountTypeAndBranch(
                AccountType.FEE_INCOME, branch, startDate, endDate
            );
        pl.setTotalRevenue(interestIncome.add(feeIncome));
        
        // Expenses
        BigDecimal financialExpenses = accountingEntryRepository
            .sumDebitsByAccountTypeAndBranch(
                AccountType.FINANCIAL_EXPENSE, branch, startDate, endDate
            );
        BigDecimal operatingExpenses = accountingEntryRepository
            .sumDebitsByAccountTypeAndBranch(
                AccountType.OPERATING_EXPENSE, branch, startDate, endDate
            );
        pl.setTotalExpenses(financialExpenses.add(operatingExpenses));
        
        // Net income
        pl.setNetIncome(pl.getTotalRevenue().subtract(pl.getTotalExpenses()));
        
        // ROA (Return on Assets)
        BigDecimal totalAssets = accountingEntryRepository
            .sumBalancesByAccountTypeAndBranch(
                AccountType.ASSET, branch, endDate
            );
        pl.setReturnOnAssets(
            pl.getNetIncome().divide(totalAssets, 4, RoundingMode.HALF_UP)
        );
        
        return pl;
    }
}
```

### Branch Setup Checklist

When setting up a new branch:

1. **Create Branch Record**
   - Unique name
   - Complete address information
   - Contact details

2. **Configure Custom Fields**
   - Manager information
   - Operating hours
   - Geographic coordinates

3. **Set Up Accounts**
   - Cash account
   - Loan portfolio account
   - Savings liability account
   - Expense accounts

4. **Assign Users**
   - Branch manager
   - Loan officers
   - Tellers
   - Accountant

5. **Configure Products**
   - Enable loan products for branch
   - Set up savings products
   - Configure fee structures

6. **Test Operations**
   - Create test client
   - Process test loan
   - Verify accounting entries

### Scenario 2: Branch Consolidation

```java
// Merge smaller branch into larger one
@Transactional
public void consolidateBranches(Long sourceBranchId, Long targetBranchId) {
    Branch source = branchService.findOne(sourceBranchId).get();
    Branch target = branchService.findOne(targetBranchId).get();
    
    // Move all profiles
    profileRepository.findByBranch(source)
        .forEach(profile -> {
            profile.setBranch(target);
            profileRepository.save(profile);
        });
    
    // Move all users
    userRepository.findByBranch(source)
        .forEach(user -> {
            user.setBranch(target);
            userRepository.save(user);
        });
    
    // Transfer account balances
    transferBranchBalances(source, target);
    
    // Archive source branch
    source.setName(source.getName() + " (Closed)");
    branchService.update(source);
}
```
