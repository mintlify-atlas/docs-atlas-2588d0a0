## Overview

The Bonds API enables financial institutions to manage bond issuances and track bond investments. Bonds represent debt securities issued by the institution or held as investments, with defined maturity dates, coupon payments, and interest rates.

## Bonds

### Create Bond

Create a new bond in IN_PROGRESS status.

<CodeGroup>
```bash cURL
curl -X POST https://api.opencbs.com/api/bonds \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "bondProductId": 5,
    "profileId": 789,
    "bankAccountId": 123,
    "equivalentCurrencyId": 1,
    "isin": "US0378331005",
    "number": 1000,
    "interestRate": 4.5,
    "penaltyRate": 2.0,
    "valueDate": "2024-01-15",
    "couponDate": "2024-07-15",
    "expireDate": "2029-01-15",
    "frequency": "SEMI_ANNUAL",
    "maturity": 60,
    "interestScheme": "FLAT",
    "sellDate": "2024-01-15"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.opencbs.com/api/bonds', {
  method: 'POST',

    'Authorization': 'Bearer YOUR_TOKEN',
```

### Roll Back Bond Events

<CodeGroup>
```bash cURL

  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "comment": "Reversing incorrect coupon payment"
  }'
```

```javascript JavaScript

  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
```

</CodeGroup>

  Bond ID
</ParamField>

## Permissions

The following permissions control access to bond operations:

- `BONDS` - View bond accounts
- `BONDS_CREATE` - Create new bonds
- `BONDS_UPDATE` - Update bonds in IN_PROGRESS status
- `BONDS_SELL` - Start/activate bonds
- `ACTUALIZE_BONDS` - Trigger bond actualization
- `ROLLBACK_BOND_EVENTS` - Roll back bond transactions
- `VALUE_DATE_BOND_EVENTS` - Modify bond value dates

## Common Use Cases

### Issuing Corporate Bonds

1. **Create bond**: POST to `/api/bonds` with bond details
2. **Preview schedule**: Verify coupon payment dates and amounts
3. **Start bond**: POST to `/api/bonds/start/{id}` to activate
4. **Process coupons**: Use repayment endpoints to record coupon payments
5. **Track events**: Monitor all transactions via events endpoint

### Bond Investment Tracking

1. **Create bond**: Record bond purchase with ISIN and details
2. **Set value date**: Establish when the bond becomes active
3. **Actualize regularly**: Keep accrued interest up to date
4. **Record coupons**: Process incoming coupon payments
5. **Monitor to maturity**: Track bond until expiration date

### Currency Conversion

When bonds are denominated in foreign currencies:
1. Set `equivalentCurrencyId` to your base currency
2. Use `/api/bonds/convert-amount` to get current valuations
3. Actualization will handle exchange rate fluctuations
