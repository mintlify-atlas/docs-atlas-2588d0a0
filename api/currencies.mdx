The Currencies API provides endpoints for retrieving currency information. Currencies are used throughout the system for loans, savings, accounting, and reporting.

## Get All Currencies

<CodeGroup>
```http Request
GET /api/currencies
```
</CodeGroup>

Retrieves a list of all currencies configured in the system.

### Response

Returns an array of currency objects.

<ResponseField name="array" type="array">
  Array of currency objects
  <Expandable title="Currency Object">
    <ResponseField name="id" type="long">
      Unique identifier for the currency
    </ResponseField>
    <ResponseField name="name" type="string">
      Full name of the currency (e.g., "US Dollar", "Euro")
    </ResponseField>
    <ResponseField name="code" type="string">
      ISO 4217 currency code (e.g., "USD", "EUR", "GBP")
    </ResponseField>
    <ResponseField name="isMain" type="boolean">
      Indicates whether this is the main/base currency for the system. Only one currency should have this flag set to true.
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```json Response Example

  {
    "id": 1,
    "name": "US Dollar",
    "code": "USD",
    "isMain": true
  },
  {
    "id": 2,
    "name": "Euro",
    "code": "EUR",
    "isMain": false
  },
  {
    "id": 3,
    "name": "British Pound",
    "code": "GBP",
    "isMain": false
  },
  {
    "id": 4,

  }
]
```
</CodeGroup>

## Get Currencies for Lookup

<CodeGroup>
```http Request
GET /api/currencies/lookup
```
</CodeGroup>

Retrieves currencies with search functionality and pagination, typically used for dropdown/lookup components in the UI.

### Query Parameters

<ParamField query="search" type="string">
  Search string to filter currencies by name or code
</ParamField>

<ParamField query="page" type="integer" default="0">
  Page number for pagination (zero-based)
</ParamField>

<ParamField query="size" type="integer" default="20">
  Number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sort criteria in the format: `property,asc|desc` (e.g., "code,asc" or "name,desc")
</ParamField>

### Response

<ResponseField name="content" type="array">
  Array of currency objects matching the search criteria
  <Expandable title="Currency Object">
    <ResponseField name="id" type="long">
      Unique identifier for the currency
    </ResponseField>
    <ResponseField name="name" type="string">
      Full name of the currency
    </ResponseField>
    <ResponseField name="code" type="string">
      ISO 4217 currency code
    </ResponseField>
    <ResponseField name="isMain" type="boolean">
      Whether this is the main currency
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="totalElements" type="long">
  Total number of currencies matching the search criteria
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages available
</ResponseField>

<ResponseField name="size" type="integer">
  Number of items per page
</ResponseField>

<ResponseField name="number" type="integer">
  Current page number (zero-based)
</ResponseField>

## Usage Examples

### Search for Euro currency

<CodeGroup>
```bash cURL
curl -X GET "https://api.opencbs.com/api/currencies/lookup?search=EUR" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.opencbs.com/api/currencies/lookup?search=EUR',
  {
    headers: {
      'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
    }
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.opencbs.com/api/currencies/lookup',
    params={'search': 'EUR'},
    headers={'Authorization': 'Bearer YOUR_ACCESS_TOKEN'}
)
data = response.json()
print(data)
```
</CodeGroup>

### Get all currencies (unpaginated)

<CodeGroup>
```bash cURL
curl -X GET "https://api.opencbs.com/api/currencies" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

```javascript JavaScript
const response = await fetch(
  'https://api.opencbs.com/api/currencies',
  {
    headers: {
      'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
    }
  }
);
const currencies = await response.json();

// Find the main currency
const mainCurrency = currencies.find(c => c.isMain);
console.log('Main currency:', mainCurrency.code);
```

```python Python
import requests

response = requests.get(
    'https://api.opencbs.com/api/currencies',
    headers={'Authorization': 'Bearer YOUR_ACCESS_TOKEN'}
)
currencies = response.json()

# Find the main currency
main_currency = next(c for c in currencies if c['isMain'])
print(f"Main currency: {main_currency['code']}")
```
</CodeGroup>

## Currency Properties

### ISO 4217 Currency Codes

The `code` field uses standard ISO 4217 three-letter currency codes. Common examples:

- **USD** - United States Dollar
- **EUR** - Euro
- **GBP** - British Pound Sterling
- **JPY** - Japanese Yen
- **CHF** - Swiss Franc
- **CAD** - Canadian Dollar
- **AUD** - Australian Dollar
- **CNY** - Chinese Yuan
- **INR** - Indian Rupee
- **BRL** - Brazilian Real

### Main Currency

The `isMain` flag identifies the base currency for the system. This currency is typically used for:

- Default currency for new accounts
- Internal reporting and analytics
- Exchange rate calculations (other currencies are expressed relative to the main currency)
- System-level financial operations

Only one currency in the system should have `isMain` set to `true`.

## Notes

- Currencies are typically configured during system setup and rarely modified
- The currency list is read-only via the API; currency configuration is done through system administration
- All monetary amounts in the API reference their currency using the currency ID or code
- When creating loans, savings accounts, or other financial products, you must specify a valid currency ID from this list
