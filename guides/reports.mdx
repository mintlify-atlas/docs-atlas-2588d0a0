OpenCBS Cloud provides comprehensive reporting capabilities to analyze operations, track performance, and generate regulatory reports. This guide covers how to access, generate, and customize reports.

## Understanding Reports

Reports in OpenCBS Cloud fall into several categories:

- **Operational Reports**: Daily operations and transactions
- **Financial Reports**: Balance sheets, income statements, trial balances
- **Portfolio Reports**: Loan and savings portfolio analysis
- **Regulatory Reports**: Compliance and regulatory submissions
- **Management Reports**: Performance metrics and KPIs
- **Custom Reports**: Institution-specific reports

## Accessing Reports

<Steps>
  <Step title="Navigate to Reports">
    From the main menu, click **Reports** to access the reports list.
  </Step>

  <Step title="Browse Available Reports">
    The reports page displays all available reports organized by category. Each report shows:
    
    - Report name and description
    - Report category
    - Available output formats
    - Required permissions
    
    Use the search box to find specific reports by name.
  </Step>

  <Step title="View Report Details">
    Click on a report to see:
    - Full description
    - Required parameters
    - Output format options
    - Estimated run time
  </Step>
</Steps>

## Generating Reports

<Steps>
  <Step title="Select Report">
    Click on the report you want to generate from the reports list. This opens the report parameter modal.
  </Step>

  <Step title="Configure Report Parameters">
    Enter the required parameters for the report:
    
    <Tabs>
      <Tab title="Date Parameters">
        Most reports require date parameters:
        
        **Single Date Reports**
        - Balance Sheet (as of date)
        - Trial Balance (as of date)
        - Portfolio Summary (specific date)
        
        **Date Range Reports**
        - Income Statement (from/to)
        - Transaction Reports (from/to)
        - Loan Disbursements (from/to)
        
        <Note>
        **Start Date**: Defaults to first day of current month
        
        **End Date**: Defaults to current date
        
        You can modify these dates as needed for your reporting period.
        </Note>
      </Tab>

      <Tab title="Filter Parameters">
        Narrow report scope with filters:
        
        **Branch**
        - Select specific branch
        - Leave blank for all branches (consolidated)
        
        **Product**
        - Loan product filter
        - Savings product filter
        - Specific to product-related reports
        
        **Account**
        - Specific account selection
        - For account-specific reports (ledgers, statements)
        
        **Status**
        - Active/Inactive/Closed
        - Loan status filters
        - Account status filters
        
        **Currency**
        - Filter by currency
        - For multi-currency institutions
      </Tab>

      <Tab title="Lookup Parameters">
        Some reports use lookup fields:
        
        **Profile Lookup**
        - Search for specific client
        - Enter name or ID
        - Select from search results
        
        **Account Lookup**
        - Search for GL account
        - Can search by account code or name
        - Shows account hierarchy
        
        **User Lookup**
        - Select specific loan officer or user
        - For user-specific reports
        
        <Tip>
        Lookup fields support partial text search. Type a few characters to see matching options.
        </Tip>
      </Tab>
    </Tabs>
  </Step>

  <Step title="Select Output Format">
    Choose the report format:
    
    **Excel (XLSX)**
    - Spreadsheet format
    - Editable and analyzable
    - Best for data manipulation
    - Default for most reports
    
    **HTML (Web Preview)**
    - View in browser
    - Interactive navigation
    - Print from browser
    - Best for quick review
    
    **PDF**
    - Formatted for printing
    - Non-editable
    - Best for official documentation
    - Available for select reports
    
    <Note>
    Format availability depends on the report type. Most operational reports generate as Excel, while some financial reports support multiple formats.
    </Note>
  </Step>

  <Step title="Generate Report">
    Click **Generate** to run the report. The system:
    
    - Validates all required parameters
    - Queries the database
    - Formats the output
    - Generates the file
    
    **For Excel Reports**:
    - Download starts automatically
    - File saved to your downloads folder
    - Opens in Excel or spreadsheet app
    
    **For HTML Reports**:
    - Opens in new browser tab
    - Can print from browser
    - Navigate using browser controls
    
    <Tip>
    Large reports may take longer to generate. The system shows a loading indicator while processing. For very large datasets, consider narrowing the date range or filters.
    </Tip>
  </Step>
</Steps>

## Common Reports

### Loan Reports

**Loan Portfolio Summary**
- Total outstanding balance
- Number of active loans
- Portfolio at Risk (PAR)
- Breakdown by product, branch, status

**Loan Disbursement Report**
- All loans disbursed in period
- Disbursement amounts
- Clients and loan officers
- Product breakdown

**Loan Repayment Report**
- Repayments collected in period
- Principal and interest breakdown
- Collection efficiency
- By loan officer or branch

**Loan Aging Report**
- Days Past Due (DPD) analysis
- Aging buckets (0-30, 31-60, 61-90, 90+)
- Outstanding balances by aging
- Portfolio at Risk indicators

**Loans Approaching Maturity**
- Loans maturing soon
- Maturity dates
- Outstanding balances
- Renewal candidates

### Savings Reports

**Savings Balance Report**
- Current balances all accounts
- Breakdown by product
- Total deposits
- Branch analysis

**Savings Transaction Report**
- Deposits and withdrawals in period
- Transaction volumes
- Average transaction sizes
- Active vs. dormant accounts

**Interest Posted Report**
- Interest posted to accounts
- By period and product
- Total interest paid
- Interest rates analysis

**Dormant Accounts Report**
- Accounts with no activity
- Inactivity period
- Balance amounts
- Potential closures

### Financial Reports

**Balance Sheet**
- Assets, Liabilities, Equity
- As of specific date
- Comparative periods
- Branch or consolidated

**Income Statement (P&L)**
- Revenue and expenses
- For specific period
- Net income calculation
- Period comparisons

**Trial Balance**
- All accounts with balances
- Debit and credit totals
- Verification of balance
- Account hierarchy

**General Ledger**
- Transaction detail by account
- Running balances
- All entries in period
- Audit trail

**Cash Flow Statement**
- Cash inflows and outflows
- Operating, investing, financing activities
- Period analysis
- Liquidity indicators

### Portfolio Analysis Reports

**Portfolio at Risk (PAR)**
- Loans with overdue payments
- PAR 1, 30, 60, 90 day calculations
- By branch, product, officer
- Trend analysis

**Portfolio Quality Indicators**
- Default rates
- Write-off ratios
- Recovery rates
- Risk ratings

**Loan Officer Performance**
- Disbursements by officer
- Portfolio size
- Collection rates
- PAR by officer

**Product Performance**
- Performance by loan product
- Outstanding balances
- Delinquency rates
- Profitability analysis

### Operational Reports

**Daily Transaction Report**
- All transactions for a day
- By transaction type
- User activity
- Branch operations

**User Activity Report**
- User login history
- Actions performed
- Transaction volumes
- Audit trail

**Exception Report**
- Failed transactions
- System errors
- Unusual activities
- Items requiring attention

## Working with Generated Reports

### Excel Reports

Once downloaded, you can:

**Analyze Data**
- Use Excel formulas and functions
- Create pivot tables
- Generate charts and graphs
- Apply filters and sorting

**Customize Presentation**
- Format cells and columns
- Add company branding
- Adjust column widths
- Apply conditional formatting

**Share and Export**
- Save in various formats
- Email to stakeholders
- Print formatted versions
- Archive for record keeping

<Tip>
Excel reports often include multiple worksheets. Check all tabs for complete data, summary sheets, and notes.
</Tip>

### HTML Reports

When viewing HTML reports:

**Navigation**
- Use browser back/forward buttons
- Scroll for long reports
- Use browser search (Ctrl+F) to find specific items

**Printing**
- Use browser print function (Ctrl+P)
- Adjust print settings for best results
- Consider landscape for wide reports
- Use print preview to verify formatting

**Saving**
- Use browser "Save Page As" to save HTML
- Convert to PDF using browser print-to-PDF
- Take screenshots for quick sharing

## Scheduling Reports

Some institutions can configure scheduled reports:

**Scheduled Report Features**
- Automatic generation on schedule (daily, weekly, monthly)
- Email delivery to specified recipients
- Pre-configured parameters
- Reduces manual effort

<Note>
Report scheduling is configured by administrators. Contact your system administrator if you need regular automated reports.
</Note>

## Custom Report Requests

If you need a report not available in the standard list:

<Steps>
  <Step title="Document Requirements">
    Clearly define what you need:
    - Data required
    - Filters and parameters
    - Output format
    - Frequency needed
    - Business purpose
  </Step>

  <Step title="Submit Request">
    Contact your system administrator or report developer with requirements. Provide:
    - Sample output (if available)
    - Data sources needed
    - Calculations required
    - Distribution list
  </Step>

  <Step title="Review and Test">
    When custom report is available:
    - Test with real data
    - Verify calculations
    - Confirm parameters work correctly
    - Validate output format
  </Step>

  <Step title="Documentation">
    Document the custom report:
    - Purpose and use case
    - Parameter definitions
    - How to interpret results
    - Update frequency
  </Step>
</Steps>

## Common Report Parameters Explained

### Date Parameters

**As of Date**
- Specific point in time
- Snapshot of balances/status at that date
- Used for Balance Sheet, Portfolio Summary

**Date Range (From/To)**
- Period of time
- Captures transactions/activity during period
- Used for Income Statement, Transaction Reports

**Default Dates**
- Start Date: Usually first day of current month
- End Date: Usually current date
- Can be modified as needed

### Filter Parameters

**Branch**
- Filter results to specific branch
- Leave blank for all branches
- Consolidated reports use "All Branches"

**Product**
- Filter to specific loan or savings product
- Shows only accounts/loans for that product
- Useful for product performance analysis

**Status**
- Active, Inactive, Closed, Pending
- Filter by current status
- Specific to report type

## Troubleshooting Reports

<Warning>
**Common Report Issues:**

**"No data found"**
- Check date range is correct
- Verify filter parameters
- Ensure data exists for selected criteria
- Try broader date range or fewer filters

**"Report generation timeout"**
- Report too large for selected parameters
- Narrow date range
- Add more specific filters
- Run during off-peak hours

**"Download failed"**
- Check browser download settings
- Disable popup blockers
- Try different browser
- Check available disk space

**"Incorrect totals"**
- Verify date parameters
- Check filter selections
- Confirm understanding of report calculations
- Contact administrator if calculations appear wrong
</Warning>

## Report Permissions

Access to reports is controlled by user permissions:

- Users see only reports they have permission to access
- Some reports restricted to specific roles (Manager, Accountant, Admin)
- Branch users may see only their branch data
- Sensitive reports require additional approvals

<Note>
If you need access to a report that's not visible, contact your system administrator to request appropriate permissions.
</Note>

## Regulatory Reporting

For regulatory compliance reporting:

**Central Bank Reports**
- Configured for specific regulatory requirements
- Usually scheduled monthly or quarterly
- Often require approval before submission
- May need multiple approver signatures

**Prudential Reports**
- Capital adequacy
- Liquidity ratios
- Asset quality
- Regulatory compliance metrics

**Audit Reports**
- Transaction audit trails
- User activity logs
- System change logs
- Financial statement audit packs

<Warning>
Regulatory reports often have strict deadlines and format requirements. Run test reports in advance and verify all data before submitting official reports.
</Warning>
